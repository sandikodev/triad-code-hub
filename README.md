## ğŸš€ Memulai (Quick Start)

Repositori ini dioptimalkan untuk performa tinggi menggunakan **Bun** atau **pnpm**.

1.  **Clone repositori:**
    ```bash
    git clone https://github.com/your-username/triad-code-hub.git
    cd triad-code-hub
    ```

2.  **Instal dependensi:**
    ```bash
    # Direkomendasikan
    bun install
    # atau
    pnpm install
    ```

3.  **Setup API Key:**
    Dapatkan API Key dari [Google AI Studio](https://aistudio.google.com/) dan atur di environment Anda:
    ```bash
    export API_KEY=your_gemini_api_key_here
    ```

4.  **Jalankan development server:**
    ```bash
    bun dev
    # atau
    pnpm dev
    ```
