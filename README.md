## ğŸ›ï¸ Mengapa "The Triad"?

Dunia perangkat lunak sedang bergeser. Kami percaya masa depan dibangun di atas tiga pilar:
1.  **Zig:** Untuk presisi manual dan kontrol memori total tanpa *hidden allocations*.
2.  **Rust:** Untuk keamanan memori tanpa *garbage collector* dan abstraksi tanpa biaya.
3.  **Elixir:** Untuk skalabilitas masif dan sistem yang mampu memulihkan diri sendiri (*self-healing*).

## ğŸš€ Memulai (Quick Start)

Ingin menjalankan lab ini di mesin lokal Anda? Sangat mudah:

1.  **Clone repositori ini:**
    ```bash
    git clone https://github.com/your-username/triad-code-hub.git
    cd triad-code-hub
    ```

2.  **Instal dependensi:**
    ```bash
    npm install
    ```

3.  **Setup API Key:**
    Dapatkan API Key dari [Google AI Studio](https://aistudio.google.com/) dan atur sebagai environment variable:
    ```bash
    export API_KEY=your_gemini_api_key_here
    ```

4.  **Jalankan development server:**
    ```bash
    npm run dev
    ```
